Para destacarmos um trecho como código usamos a Tag CODE.

        
    num = int(input('Digite um número'))
    if num % 2 == 0;
        print(f'O número {num} é PAR')
    else;
        print(f'O número {num} é ÍMPAR')
    print('Fim do programa.')

Para deixarmos o texto como esta acima, usamos a TAG PRE, pois a mesma considera todos os espaçamentos usados, e coloquei a TAG CODE dentro da TAG PRE.
